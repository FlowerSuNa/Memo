# nbextensions

jupyter notebook 확장 프로그램

### Install

1. pip

```linux
pip install jupyter_contrib_nbextensions
```

```linux
jupyter contrib nbextension install --user
```

2. conda

```linux
conda install -c conda-forge jupyter_contrib_nbextensions
```

```linux
jupyter contrib nbextension install --user
```
